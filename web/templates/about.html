About This Project

An end-to-end machine learning project for spam detection

🎯 Project Overview

The Spam Email Detector is a machine learning application that uses Natural Language Processing (NLP) to classify emails as either spam or ham (legitimate). The system is trained on thousands of labeled emails and uses multiple classification algorithms to achieve high accuracy.

🤖 Machine Learning Models

The project implements and compares several ML algorithms:

  • Naive Bayes - Fast and effective baseline
  • Logistic Regression - Simple but powerful linear classifier
  • Support Vector Machine (SVM) - State-of-the-art performance
  • Random Forest - Ensemble learning with feature importance
  • Gradient Boosting - Advanced boosting algorithm

📊 Dataset

This project uses the SMS Spam Collection Dataset from Kaggle, containing 5,574 labeled messages with a balanced distribution of spam and ham messages.

5,574 Total Messages
4,827 Ham Messages
747 Spam Messages
